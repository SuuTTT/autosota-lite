## 9. Recommended submission routing for structural-entropy / graph / RL work

### 9.1 If the contribution is mainly graph mining / community detection

Best route:

1. **ICDM 2026** if ready now.
2. **WSDM 2027** if web/social/network graph angle exists.
3. **KDD 2027** if official CFP opens and the contribution is strong enough.
4. **AAAI 2027** if AI/planning/optimization framing is stronger than data-mining framing.
5. **Information Sciences SI: Graph-based solutions for AI** as a journal/special-issue route.

### 9.2 If the contribution is graph systems / scalable graph computation

Best route:

1. **PVLDB 2027 monthly rounds**
2. **ICDE 2027**
3. **SIGMOD 2027 R3/R4**
4. **IEEE Transactions on Computers SI: Graph-Centric Computing**

Paper must emphasize:
- scalability,
- indexing/query/data model,
- system design,
- distributed/parallel graph computation,
- reproducible benchmark pipeline.

### 9.3 If the contribution is RL / world model / planning

Best route:

1. **AAAI 2027** for broad AI.
2. **ICLR/ICML/NeurIPS 2027** once CFPs are official.
3. **CoRL/ICRA/IROS/RSS** only with robotics experiments.
4. **AIIDE/CoG** if tested in game environments.

### 9.4 If the contribution is multi-agent swarm / UAV / robotic network

Best route:

1. **CoRL 2026** if ready immediately.
2. **ICRA 2027 / IROS 2027 / RSS 2027** after robotics validation.
3. **IEEE TNSE low-altitude intelligent networks SI** if UAV network framing is strong.
4. **IEEE TNSE autonomous agent systems SI** for mission-centric coordination.

### 9.5 If the contribution is biological networks

Best route:

1. **Bioinformatics / TCBB / Briefings in Bioinformatics** if the biological problem is central.
2. **ISMB / RECOMB / BIBM** when next CFP opens.
3. **MICCAI / ISBI** only if imaging/clinical multimodal data is central.

### 9.6 If the contribution is game AI

Best route:

1. **AIIDE 2026** if ready by June 2026.
2. **IEEE CoG 2027 / FDG 2027 / CHI PLAY 2027** when CFPs open.
3. **IEEE Transactions on Games** for mature journal version.

### 9.7 If the contribution is music/audio

Best route:

1. **DCASE 2026 Workshop/Challenge** for sound-event/acoustic-scene work.
2. **ICASSP 2027** for signal/audio/speech model.
3. **ISMIR 2027** for music information retrieval.
4. **TASLP / TMM / TOMM** for mature journal version.
