================================================================================
KNOWN LIMITATIONS
================================================================================

1. Email: Gmail only (not generic SMTP)
   └─ Requires app-specific password, not regular Gmail password
   └─ 2-Step Verification must be enabled

2. ADC: No support for service account key files
   └─ By design (easier for local dev)
   └─ Service accounts use OIDC federation or ADC config

3. Secrets: Read-only operations
   └─ Create/update/delete via gcloud CLI only
   └─ Prevents accidental modifications in code

4. No secret versioning in code
   └─ GCP handles versions automatically
   └─ Code always uses 'latest' version
